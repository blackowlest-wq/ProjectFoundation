# 作業記録テンプレート

## 目的

作業記録に残す項目を揃え、後から変更理由、確認内容、未対応事項、再確認条件を追跡できるようにする。

## 使い方

- 作業完了時だけでなく、判断が分かれた時点でも必要に応じて記録する。
- テスト未実行、規約未確認、指摘保留がある場合は理由と再確認条件を必ず残す。
- ユーザから追加指摘を受けた場合は、規約・レビュー観点・チェック表へ反映したかを記録する。
